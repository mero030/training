# 🏋️ Training Tracker

A personal Progressive Web App (PWA) for logging sets, reps, and tracking workout progress — built around a 4-day training program.

**Works offline. Installs like a native app. No account needed.**

---

## 📱 How to Install on Your Phone

### Step 1 — Enable GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under **Source**, select **Deploy from a branch**
5. Set branch to **main**, folder to **/ (root)**
6. Click **Save**
7. Wait ~60 seconds, then your app is live at:

```
https://YOUR-USERNAME.github.io/YOUR-REPO-NAME/
```

### Step 2 — Add to Home Screen (iPhone)

1. Open the URL above in **Safari** (must be Safari on iPhone)
2. Tap the **Share** button (box with arrow at bottom of screen)
3. Scroll down and tap **"Add to Home Screen"**
4. Name it **Training** and tap **Add**
5. The app icon appears on your home screen ✓

### Step 3 — Add to Home Screen (Android)

1. Open the URL in **Chrome**
2. Tap the **three dots** menu (top right)
3. Tap **"Add to Home Screen"**
4. Tap **Add**
5. The app icon appears on your home screen ✓

---

## ✅ Features

- All 4 training days pre-loaded (Upper / Lower / Mobility / Full Body)
- Log sets with reps, weight/hold time, and notes
- Mark sets as complete with one tap
- Session progress bar
- Full session history
- Works completely **offline** after first load
- Data saved locally on your device
